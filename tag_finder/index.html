Info

The Tag Finder Gadget searches for files and folders that have a particular tag associated with them.

Usage

Use the dropdown to search for and select the tag you want to find files and folders associated with..

If you only want to search certain folders on your site, you can choose a subfolder via the "Choose Subfolder" option. Otherwise, the gadget will search the entire site.
